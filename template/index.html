The Tomato Leaf Disease Predictor is a flask web application which classifies a plant/leaf image into 10 categories viz. 'Tomato_mosaic_virus', 'Early_blight', 'Septoria_leaf_spot', 'Bacterial_spot', 'Target_Spot', 'Spider_mites Two spotted_spider_mite', 'Tomato_Yellow_Leaf_Curl_Virus', 'Late_blight', 'Healthy', and 'Leaf_Mold'. The code is written in Python 3.7.1 and makes use of Keras and Tensorflow libraries in developing an InceptionV3 based image classification web application.
